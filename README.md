# Code-Quiz

This is a quiz containing eight JavaScript questions. Players have three minutes to complete the quiz. One point is added for each question answered correctly. If a question is answered incorrectly 25 seconds will be deducted from the time. Players can save their high scores and keep adding new high score to the list or clear the list if they so choose.
